BTS SIO SLAMとは何ですか？

BTS SIOオプションSLAMの学生は、3つの活動分野のスキルを習得します。そのうちの2つは、BTSSIOオプションSISRと共通しています。


- ITサービスのサイバーセキュリティ：学生は、技術的、組織的、法的側面を考慮に入れて、サイバーセキュリティとその企業への統合についてトレーニングを受けています...したがって、彼はデータの保護と企業のデジタルIDのスキルを習得し、機器とユーザーの使用。 SLAMオプション：彼は、アプリケーションソリューションのサイバーセキュリティとその開発について訓練を受けています。


- ITサービスのサポートと提供：学生は、既存のITサービスの可用性を確保することにより、ユーザーの期待に応えることができます。 彼はまた、企業のITニーズを考慮に入れ、エンプロイアビリティを維持しながらITサービスのデジタルトランスフォーメーションをサポートできるようになります。 したがって、IT資産を管理し、インシデントに対応し、Web上で会社の存在感を高め、独自の専門能力開発を組織するスキルを習得します。


- SLAMオプション：アプリケーションの設計と開発：学生は、アプリケーションソリューションのソフトウェアコンポーネントの設計、開発、展開、および保守に参加します。 したがって、彼は、アプリケーションソリューションの設計と開発、その保守だけでなく、データベースとデジタルデータ管理のスキルも習得します。
